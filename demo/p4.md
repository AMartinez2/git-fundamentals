1. Make a new directory and initialize inside
```
mkdir test
cd test
git init
```

2. Reset previous step
```
cd ..
rm -rf test
```

3. Create an empty directory using git
```
git init test
```

4. Remove the git directory
```
cd test
rm -rf .git
```

5. Clone this repo
```
git clone https://github.com/donaldong/git-fundamentals.git
```
